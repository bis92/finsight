# PRD: FinSight

> 이 문서(PRD)와 `ARCHITECTURE.md`·`ADR.md`가 함께 계획의 **단일 원천(source of truth)**이다. 세부 UI 규약은 `UI_GUIDE.md`를 따른다.

## 목표

한국 개인 소비자가 카드 명세서·은행 거래내역 CSV를 올리면, Claude가 컬럼을 자동 인식해 표준 스키마로 정규화하고 소비 대시보드를 보여준다. Pro에서는 Opus 4.8이 **지출 진단 리포트**를 생성해 "내 돈이 어디로 갔고, 무엇을 바꿔야 하는지"를 즉시 파악하게 한다. 엑셀 수작업이 필요 없다.

## 사용자

여러 카드·계좌 지출을 한눈에 정리하고 싶은 한국 개인(B2C 가계부/소비분석). 국내 카드사(신한·삼성·현대 등) 명세서를 CSV로 내려받아 쓰며, 원화·한글 가맹점명 기준. 은행/카드사 앱 분석이 계좌별로 파편화돼 통합 뷰가 없다고 느끼는 사람.

## 핵심 기능

1. **CSV 업로드 + 컬럼 자동매핑** — 인코딩 감지(UTF-8/EUC-KR) → 파싱 → LLM이 헤더+샘플 ≤20행으로 컬럼 매핑 → 사용자 매핑 확인 스텝. 국내 카드사마다 다른 컬럼/인코딩/날짜형식을 하드코딩 없이 흡수.
2. **Free 대시보드(사실 조회)** — 이번 달 카테고리별 지출, 상위 가맹점, 총지출/수입(순지출), 기본 차트(파이·바). 상세는 SideView, 재분류는 Modal. LLM은 컬럼 매핑에만 쓰고, 집계는 코드로 계산.
3. **Pro 지출 진단 리포트(해석·코칭)** — Opus 4.8이 집계 결과를 근거로 서술형 진단문·절감 제안을 구조화 JSON으로 생성 + **정기구독 후보 탐지**. 페이월은 서버 게이팅(`profiles.plan`).
4. **게스트 데모** — 가입 없이 샘플 fixture로 대시보드를 읽기전용 체험 → "내 파일로 해보기" CTA로 가입 유도.
5. **인증** — 카카오 + 구글 OAuth (Supabase Auth). 이메일/비밀번호 없음.
6. **결제 업그레이드** — Free→Pro 구독. phase 0은 게이팅 구조만(mock), 실결제 연동은 phase 1 (기본 Polar, ADR-007 참고).

## 과금 모델 (ADR-006)

기능 **깊이**로 가른다(수익 게이팅은 횟수가 아님).

- **Free** = "내가 뭘 썼나"(집계, LLM 가벼움). 대시보드 기능 전부. 단, 업로드는 **월 5회**까지 — 수익 게이팅이 아니라 매핑 LLM 비용 폭주를 막는 안전장치.
- **Pro** = "그래서 뭘 해야 하나"(LLM 추론, 무거움). 업로드·분석 무제한. Pro 앵커 2개: **지출 진단 리포트**, **정기구독 후보 탐지**.
- 단일 파일에 여러 달 거래가 있으면 merchant+amount+cadence로 반복 후보 탐지, 단일 월만 있으면 패턴 기반 "후보"로 표시. **확정 표현 금지.**

## MVP 제외 사항 (로드맵)

- 은행/카드 API 직접연동(오픈뱅킹·스크래핑)
- 다중파일 병합·중복감지 (MVP는 **단일 파일 업로드**. 단, 한 CSV에 여러 달 포함 가능)
- 다월(MoM) 비교/추이 UX, PDF 리포트
- 국내 PG(토스페이먼츠/포트원) 연동 — phase 1에서 Polar와 함께 재검토(ADR-007)
- 팀/멀티유저·예산설정, 일반 API rate limit/스케일링, 모바일 앱, i18n, 다크모드

## 성공 기준

- **Activation:** 신규 사용자가 샘플 데모 또는 자기 CSV로 3분 안에 첫 대시보드를 본다.
- **업로드 품질:** 국내 주요 카드사 대표 fixture 기준 파싱 성공률 95%↑, 필수 컬럼 자동매핑 성공률 80%↑. 실패 시 수동 매핑 폴백으로 업로드 전체가 막히지 않는다.
- **인사이트 가치:** Pro 화면은 "지출 진단"·"정기구독 후보"를 각각 최소 1개 이상 표시. 근거 부족 시 후보/추정 문구.
- **비용 상한:** CSV 1건 분석에서 LLM 입력은 매핑 샘플 ≤20행 + 집계/최소 필드만, LLM 호출은 기본 경로 3회 이하(mapColumns, insights, subscription candidates).
- **전환 관찰:** 게스트→가입 CTA, Free→Pro CTA, 결제 시작 클릭을 PII 없이 기록.

## ⚠️ 알려진 리스크

1. **Polar USD ↔ 한국 타깃 긴장** — 국내카드·원화 결제 제한으로 실제 결제 가능층이 좁아질 수 있음. phase 0은 mock 게이팅으로 우회하고, phase 1 실연동 시 국내 PG 전환을 재검토(ADR-007).
2. **원본 거래 저장 책임** — 민감 금융 PII. RLS·at-rest 암호화·비공개 Storage·완전삭제(삭제권) 필수(`ARCHITECTURE.md` 보안 · ADR-009).
3. **AI 매핑 신뢰도·토큰 비용** — 매핑 오류 시 사용자 확인/수정 경로가 반드시 있어야 하고, 매핑은 샘플 행만 전송해 비용을 억제(ADR-004).

## 디자인 방향

도구처럼 보이는 금융 대시보드(신뢰·가독성), 숫자가 주인공, **Light 서피스 기본**(다크모드는 출시 후). 지출↑ 빨강 / 절약↓ 초록 시맨틱 컬러. 상세는 `UI_GUIDE.md`.
