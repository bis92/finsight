# PRD: finsight

## 목표
개인이 은행·카드사에서 받은 CSV(거래내역/명세서)를 업로드하면, LLM이 자동으로 정리·분석해 **"내 돈이 어디로 갔고, 무엇을 바꿔야 하는지"**를 즉시 보여준다. 엑셀 수작업 없이.

## 사용자
- 여러 카드·계좌의 지출을 한눈에 정리하고 싶은 개인 (B2C, 가계부/소비분석).
- 은행/카드사 앱의 분석은 계좌별로 파편화돼 있어 통합 뷰가 없다고 느끼는 사람.

## 핵심 기능 (MVP)
1. **CSV 업로드 → 자동 정규화**: 은행·카드사마다 다른 컬럼(날짜/가맹점/금액/구분)을 Claude가 샘플 행으로 자동 인식해 표준 스키마로 매핑. 인코딩(EUC-KR)·금액·날짜·환불(방향)을 정규화하고, 매핑 확인 스텝으로 사용자가 확정/수정. **MVP는 단일 파일 업로드**(다중파일 병합·중복감지는 로드맵). 오분류는 재분류 Modal로 수정.
2. **기본 소비 대시보드 (Free)**: 이번 달 카테고리별 지출, 상위 가맹점, 총지출/수입, 기본 차트(파이·바).
3. **심화 인사이트 (Pro)**: LLM 기반 소비 코칭 + 정기구독·반복결제 탐지·절감 제안. (MVP Pro 앵커 2종)
4. **인증**: Supabase Auth (이메일 + 구글 소셜).
5. **구독 결제**: Polar 구독으로 Pro 해제. 웹훅으로 권한 반영.
6. **배포**: Vercel.

## Free / Pro 경계
경계는 "사용 빈도"가 아니라 **"분석 깊이"**로 가른다. (명세서는 월 1회라 횟수 제한은 무의미)

| 기능 | Free | Pro |
| --- | --- | --- |
| CSV 업로드 + 컬럼 자동매핑 | ✅ | ✅ |
| 이번 달 카테고리별 지출 / 상위 가맹점 / 총계 | ✅ | ✅ |
| 기본 차트(파이·바) | ✅ | ✅ |
| 심화 LLM 인사이트/코칭 | ❌ | ✅ |
| 정기구독·반복결제 탐지 + 절감 제안 | ❌ | ✅ |
| 다월 누적·추이 비교 (MoM) | ❌ | ✅ *(출시 후)* |
| PDF/월간 리포트 내보내기 | ❌ | ✅ *(출시 후)* |

- **MVP Pro 앵커 = 심화 LLM 인사이트 + 정기구독 탐지.** 이 둘만 MVP에 구현하고, 다월 추이·PDF는 출시 후. (다월 추이는 데이터가 2~3개월 쌓여야 가치가 나므로 초기 임팩트가 낮음)
- 논리: Free = "내가 뭘 썼나"(집계, LLM 가벼움) / Pro = "그래서 뭘 해야 하나"(LLM 추론, 무거움). 매출이 LLM 비용을 커버.

## MVP 제외 사항
- 은행/카드사 API 직접 연동(오픈뱅킹·스크래핑). CSV 업로드만.
- 다중 파일 병합·중복 거래 감지 (단일 파일 업로드로 축소).
- 다월 추이 비교, PDF 리포트 (Pro지만 출시 후).
- 팀/멀티유저·권한, 예산 설정·초과 경고.
- 트래픽 대응(rate limiting·스케일링) — 실사용 트래픽 생기면.
- 모바일 앱 (웹 반응형만). 다국어 (한국어 우선).

## 디자인
- **도구처럼 보여야 한다** — 마케팅 페이지가 아니라 매일 쓰는 금융 대시보드. 신뢰·명료함 우선.
- Light 서피스 기본, 무채색 + 절제된 단일 액센트. 보라/인디고 금지(AI 슬롭). 상세 규약은 `UI_GUIDE.md`.
